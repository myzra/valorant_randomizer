## ✨ Features

| Feature | Description |
|--------|-------------|
| 🎲 **Random Agent Generation** | Instantly roll a random agent for any player |
| 👥 **Team Generation** | Support for 1–5 players |
| 🎭 **Role Filtering** | Filter by Duelist, Initiator, Controller, or Sentinel |
| 🚫 **No Duplicates Mode** | Prevent the same agent from appearing twice in a session |
| ⚖️ **Balanced Composition** | Enforces a healthy role spread across the team |

---

## ⚖️ Balanced Composition Rules

When **Balanced Mode** is enabled:

- No two players share the same role *(except in 5-player teams)*
- In a team of 5, exactly **one role may repeat**
- Prevents extreme compositions like all-Duelist stacks
- Guarantees a playable, structured team every time

> Disable balancing to go full random mode 🔥

## 🕹️ How to Use

1. **Select** your team size (1–5 players)
2. **Configure** optional filters:
   - Role preferences per player
   - Enable/disable duplicate agents
   - Enable/disable balanced composition
3. **Click generate**
4. **Play what you get** 😄
